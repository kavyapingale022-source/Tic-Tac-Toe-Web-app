# Premium Tic-Tac-Toe Web App

A modern, highly responsive, and visually stunning Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features ✨

- **Premium UI/UX:** Clean glassmorphism aesthetic with vibrant neon highlights and a deep dark mode background.
- **Dynamic Animations:** Satisfying micro-animations for placing 'X' and 'O'.
- **Game Modes:** 
  - **Player vs Player:** Battle it out with a friend locally.
  - **Player vs AI:** Test your skills against the built-in AI opponent.
- **Score Tracking:** Automatically keeps track of X Wins, O Wins, and Draws across multiple rounds.
- **Responsive Design:** The board scales perfectly to fit both large desktop monitors and small mobile screens.

## Quick Start 🚀

To run this project locally, simply clone the repository and open the `index.html` file in any modern web browser:

```bash
git clone https://github.com/kavyapingale022-source/Tic-Tac-Toe-Web-app.git
cd Tic-Tac-Toe-Web-app
```
Then, open `index.html`.

## Deployment 🌐

This project is perfectly structured to be deployed instantly on [Vercel](https://vercel.com/). Since the main file is named `index.html`, Vercel will automatically serve it without any additional configuration.

## Design System

The styling takes heavy inspiration from futuristic spatial computing and HUDs, utilizing:
- **Sora** for bold, impactful headers.
- **Inter** for clean, legible body text.
- **JetBrains Mono** for technical data and inputs.
